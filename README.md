## About The Project
<p>Room designing game</p>
<p>Features:</p>
<ul>
    <li>3 different objects</li>
    <li>Move, rotate, delete</li>
    <li>Change colors</li>
    <li>Zoom camera</li>
    <li>Placement check</li>
    <li>Toggle grid move system</li>
    <li>Save/Load system</li>
</ul>
